# 🌍 GezginDer - İnteraktif Seyahat ve Keşif Platformu

GezginDer, kullanıcıların popüler gezi rotalarını keşfedebildiği, kendi deneyimlerini paylaşabildiği ve modern bir arayüz üzerinden etkileşime geçebildiği kapsamlı bir web uygulamasıdır.

### 🚀 Öne Çıkan Özellikler

* **Dinamik Rota Keşfi:** Şehir, Doğa ve Tarih kategorilerinde filtrelenebilir içerikler.
* **İnteraktif Harita:** Leaflet JS entegrasyonu ile harita üzerinde gezi noktaları.
* **Kullanıcı Sistemi:** Kayıt olma, giriş yapma ve profil yönetimi (Dashboard).
* **Admin Paneli:** İçerik yönetimi, kullanıcı yetkilendirme ve sistem istatistikleri.
* **Modern Tasarım:** Karanlık/Aydınlık tema desteği ve tamamen duyarlı (responsive) yapı.

### 🛠 Teknolojiler

* **Frontend:** HTML5, CSS3, JavaScript (ES6+), Leaflet.js
* **Backend:** PHP 8.x
* **Veritabanı:** SQLite (Hızlı ve taşınabilir veri yönetimi)

### 💻 Kurulum

1. Depoyu klonlayın: `git clone https://github.com/kullaniciadi/SeyahatProjesi.git`
2. PHP destekleyen bir yerel sunucuda (XAMPP vb.) çalıştırın.
3. Tarayıcıda `index.php` dosyasını açın.
